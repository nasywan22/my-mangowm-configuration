🎯 GENERAL & UTILITIES
  SUPER + Code:61      Search shortcuts (rofi, reads config)
  SUPER + Ctrl + R     Reload Waybar
  SUPER + R            Reload MangoWM configuration
  SUPER + Space        Open application launcher (rofi)
  SUPER + T            Launch terminal (kitty)

❌ EXIT / CLOSE
  SUPER + Alt + Delete Quit MangoWM
  SUPER + Q            Kill focused client

🎯 WINDOW FOCUS
  Alt + Tab            Focus next window in stack
  SUPER + Left         Focus window to the left
  SUPER + Right        Focus window to the right
  SUPER + Up           Focus window above
  SUPER + Down         Focus window below

🔄 WINDOW SWAPPING
  SUPER + Shift + Left  Swap with window on the left
  SUPER + Shift + Right Swap with window on the right
  SUPER + Shift + Up    Swap with window above
  SUPER + Shift + Down  Swap with window below

📏 WINDOW STATES
  SUPER + G            Toggle global mode
  SUPER + Tab          Toggle overview mode
  Alt + F              Toggle floating mode
  SUPER + M            Toggle maximize screen
  SUPER + F            Toggle fullscreen
  SUPER + Shift + F    Toggle fake fullscreen
  SUPER + Shift + M    Minimize window
  SUPER + Alt + I      Restore minimized windows
  SUPER + O            Toggle overlay mode
  SUPER + S            Toggle scratchpad

↕️ MOVE FLOATING WINDOW
  Ctrl + Shift + Left  Move floating window -50px left
  Ctrl + Shift + Right Move floating window +50px right
  Ctrl + Shift + Up    Move floating window -50px up
  Ctrl + Shift + Down  Move floating window +50px down

📐 RESIZE FLOATING WINDOW
  Ctrl + Alt + Left    Decrease width by 50px
  Ctrl + Alt + Right   Increase width by 50px
  Ctrl + Alt + Up      Decrease height by 50px
  Ctrl + Alt + Down    Increase height by 50px

📐 LAYOUT MANAGEMENT
  Alt + E              Set scroller proportion to 1.0
  Alt + X              Switch proportion preset
  Alt + SUPER + Ctrl + Left   Stack scroller left
  Alt + SUPER + Ctrl + Right  Stack scroller right
  Alt + SUPER + Ctrl + Up     Stack scroller up
  Alt + SUPER + Ctrl + Down   Stack scroller down
  Alt + Shift + Enter  Toggle split direction (dwindle layout)
  SUPER + N            Switch layout & refresh Waybar

🏷️ WORKSPACE (TAG) SWITCHING
  Ctrl + 1 through 9   Switch to tag 1 through 9

📦 MOVE WINDOW TO TAG
  Alt + 1 through 9    Move client to tag 1-9 and switch to it

⬅️➡️ TAG NAVIGATION
  Alt + Left           View previous tag
  Ctrl + Left          View previous tag (with client)
  Alt + Right          View next tag
  Ctrl + Right         View next tag (with client)

🔄 MOVE TAG BETWEEN MONITORS
  Ctrl + SUPER + Left  Move tag to left monitor
  Ctrl + SUPER + Right Move tag to right monitor

🖥️ MULTI-MONITOR
  Alt + Shift + Left   Focus left monitor
  Alt + Shift + Right  Focus right monitor
  SUPER + Alt + Left   Send tag to left monitor
  SUPER + Alt + Right  Send tag to right monitor

📏 GAPS
  Alt + Shift + X      Increase gaps by 1px
  Alt + Shift + Z      Decrease gaps by 1px
  Alt + Shift + R      Toggle gaps

🖱️ MOUSE BINDINGS
  SUPER + Left Click   Move window
  Any + Middle Click   Toggle maximize screen
  SUPER + Right Click  Resize window

📜 SCROLL (AXIS) BINDINGS
  SUPER + Scroll Up    View previous tag (with client)
  SUPER + Scroll Down  View next tag (with client)

🎬 ANIMATIONS
  Rofi                 Zoom animation on open/close
