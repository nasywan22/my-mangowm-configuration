🎯 GENERAL & UTILITIES
  SUPER + Code:61        Search shortcuts (rofi fuzzy search)
  SUPER + Ctrl + R       Reload Waybar
  SUPER + R              Reload MangoWM configuration
  SUPER + Space          Open application launcher (rofi)
  SUPER + T              Launch terminal (kitty)

❌ EXIT / CLOSE
  SUPER + Alt + Delete   Quit MangoWM
  SUPER + Q              Kill focused client

🎯 WINDOW FOCUS
  SUPER + Left/H         Focus window to the left
  SUPER + Right/L        Focus window to the right
  SUPER + Up/K           Focus window above
  SUPER + Down/J         Focus window below

🔄 WINDOW SWAPPING
  SUPER + Shift + Left/H   Swap with window on the left
  SUPER + Shift + Right/L  Swap with window on the right
  SUPER + Shift + Up/K     Swap with window above
  SUPER + Shift + Down/J   Swap with window below

📏 WINDOW STATES
  SUPER + G              Toggle global mode
  SUPER + Tab            Toggle overview mode
  SUPER + Alt + F        Toggle floating mode
  SUPER + M              Toggle maximize screen
  SUPER + F              Toggle fullscreen
  SUPER + Shift + F      Toggle fake fullscreen
  SUPER + Shift + M      Minimize window
  SUPER + Alt + I        Restore minimized windows
  SUPER + O              Toggle overlay mode
  SUPER + S              Toggle scratchpad

📐 LAYOUT MANAGEMENT
  SUPER + D              Toggle split direction (dwindle layout)
  SUPER + N              Switch layout & refresh Waybar
  SUPER + Ctrl + Alt + Left   Stack scroller left
  SUPER + Ctrl + Alt + Right  Stack scroller right
  SUPER + Ctrl + Alt + Up     Stack scroller up
  SUPER + Ctrl + Alt + Down   Stack scroller down

🏷️ WORKSPACE (TAG) SWITCHING
  SUPER + Ctrl + 1-9     Switch to tag 1 through 9
  SUPER + Ctrl + Left/H  View previous tag
  SUPER + Ctrl + Right/L View next tag

📦 MOVE WINDOW TO TAG
  SUPER + Alt + Shift + 1-9       Move client to tag 1-9 and switch to it
  SUPER + Ctrl + Shift + Left/H   Move client to previous tag
  SUPER + Ctrl + Shift + Right/L  Move client to next tag

🖥️ MULTI-MONITOR
  Alt + Shift + Left     Focus left monitor
  Alt + Shift + Right    Focus right monitor
  SUPER + Alt + Left     Send window to left monitor
  SUPER + Alt + Right    Send window to right monitor

↕️ MOVE FLOATING WINDOW
  SUPER + Ctrl + Shift + Up/K     Move floating window -50px up
  SUPER + Ctrl + Shift + Down/J   Move floating window +50px down
  SUPER + Ctrl + Shift + Left/H   Move floating window -50px left
  SUPER + Ctrl + Shift + Right/L  Move floating window +50px right

📐 RESIZE FLOATING WINDOW
  SUPER + Ctrl + Alt + Up/K       Decrease height by 50px
  SUPER + Ctrl + Alt + Down/J     Increase height by 50px
  SUPER + Ctrl + Alt + Left/H     Decrease width by 50px
  SUPER + Ctrl + Alt + Right/L    Increase width by 50px

📏 GAPS
  Alt + Shift + X        Increase gaps by 1px
  Alt + Shift + Z        Decrease gaps by 1px
  Alt + Shift + R        Toggle gaps

🖱️ MOUSE BINDINGS
  SUPER + Left Click     Move window
  Middle Click           Toggle maximize screen
  SUPER + Right Click    Resize window

📜 SCROLL (AXIS) BINDINGS
  SUPER + Scroll Up      View previous tag (with client)
  SUPER + Scroll Down    View next tag (with client)

🔊 AUDIO
  XF86AudioRaiseVolume   Volume up 5%
  XF86AudioLowerVolume   Volume down 5%
  XF86AudioMute          Toggle mute (speaker)
  Shift + XF86AudioMute  Toggle mute (microphone)

🔆 BRIGHTNESS
  XF86MonBrightnessUp          Brightness up 2%
  Shift + XF86MonBrightnessUp  Brightness to 100%
  XF86MonBrightnessDown        Brightness down 2%
  Shift + XF86MonBrightnessDown Brightness to 1%

⏯️ MEDIA PLAYER
  XF86AudioNext          Next track
  XF86AudioPrev          Previous track
  XF86AudioPlay          Play / Pause
